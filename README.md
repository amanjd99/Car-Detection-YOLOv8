# Real-Time Object Detection using YOLOv8

## Overview

This project performs real-time object detection using YOLOv8.

Detected Objects:
- Car
- Bike
- Bus
- Truck

---

## Technologies Used

- Python
- YOLOv8
- Google Colab
- OpenCV

---

## Training

Model:
YOLOv8n

Epochs:
50

Image Size:
640

---

## Results

mAP50:
93%

mAP50-95:
62%

---

## Features

Image Detection

Video Detection

---

## Run

Install:

pip install -r requirements.txt

python webcam_detection.py
